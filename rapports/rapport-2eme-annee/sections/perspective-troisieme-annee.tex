D'ici la fin de l'année civile, nous prévoyons :
\begin{itemize}
  \item boucler la réévaluation avec une étude sur le rejeu avec des scripts de règle
  \item il n'y aura probablement pas de place pour intégrer l'animation
  \item d'ici mars nous prévoyons d'intégrer les annotations de modèles
  \item À partir de mars le temps restant sera dédié à mon devoir d'enseignement ainsi qu'à la rédaction du manuscript.
\end{itemize}
