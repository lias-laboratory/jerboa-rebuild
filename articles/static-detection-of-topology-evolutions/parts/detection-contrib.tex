\section[Event detection]{Event detection}
\label{sec:event-detection}

Intuitively, to detect that a rule application creates (or splits, merges, …) an
orbit, one would compare the graph before and after its application. However,
comparing the members of a rule against each other allows a static detection of
topological events. Such a method presents the advantage of being independent of
the object transformed by the rule.

\subsection[Creation]{Creation}
\label{sec:creation}

A rule node matches one or more darts. So when a rule creates a node, it also
creates the corresponding darts. If the right hand side of a rule contains a
created orbit, then the application of this rule on any object creates one or
more orbits. For example, the triangulation rule
(fig.~\ref{fig:rule-triangulation}) creates the edge orbit
$R\orbit{\alpha 0, \alpha 2}(n_{1})$. Therefore, the application of this rule on
the square face (fig.~\ref{fig:before-after-house-triangulation}) creates four
edge orbits $H\orbit{\alpha 0, \alpha 2}(i)$, $H\orbit{\alpha 0, \alpha 2}(u)$,
$H\orbit{\alpha 0, \alpha 2}(q)$ and $H\orbit{\alpha 0, \alpha 2}(m)$ in $H$.

\begin{definition}[Orbit creation]
  \label{def:orbit-creation}
  Let $r : L \longrightarrow R$ be a rule, $m : L \to G$ a match morphism,
  $t: G \longrightarrow^{r,m} H$ the transformation of $G$ by $(r,m)$,
  $\orbit{o}$ an orbit type, $n$ a created node of $R$ and $d$ a dart of $H$
  matched by $n$.

  An orbit $R\orbit{o}(n)$ is \emph{created} in the rule $r$ if all of its nodes
  are created in $r$.

  An orbit $H\orbit{o}(d)$ is created by the transformation $t$ if and only if
  all of its darts are created by $t$.
\end{definition}

\begin{proposition}
  Let $r : L \longrightarrow R$ be a rule, $m : L \to G$ a match morphism,
  $t: G \longrightarrow^{r,m} H$ the transformation of $G$ by $(r,m)$,
  $\orbit{o}$ an orbit type, $n$ a created node of $R$ and $d$ a dart of $H$
  matched by $n$.

  Orbit $H\orbit{o}(d)$ is created if and only if $R\orbit{o}(n)$ is created.
\end{proposition}

\begin{proofidea}
  Due to the syntactic condition on rules, the orbit $R\orbit{o}(n)$ is
  complete. Thus, due to application of rules, $H\orbit{o}(d)$ is included in
  the image of $R\orbit{o}(n)$. Therefore, all nodes of $H\orbit{o}(d)$ are
  created. Conversely, if $H\orbit{o}(d)$ is created, then it is included in the
  image of $R\orbit{o}(n)$ and then $R\orbit{o}(n)$ is created.
\end{proofidea}

Back to the example of the face triangulation (figures
\ref{fig:rule-triangulation} and~\ref{fig:before-after-house-triangulation}), we
will follow the creation of an edge. The edge orbit
$R\orbit{\alpha 0, \alpha 2}(n_{1})$ in the triangulation rule is created
because its two nodes $n_{1}$ and $n_{2}$ are created. In the transformation,
$n_{1}$ matches 8 darts, including for example dart $i$, and $n_{2}$ also
matches 8 darts including dart $j$. The edge orbit
$H\orbit{\alpha 0, \alpha 2}(i)$ is created as well as darts $j, k, l$
(fig.~\ref{fig:house-triangulation-right-match}).


\subsection[Split]{Split}
\label{sec:split}

When a rule explicitly splits an orbit, it also splits the set of matched darts.
For example, the unsewing rule (fig.~$\ref{fig:rule-unsew-a2}$) splits the edge
orbit $L\orbit{\alpha 0, \alpha 2}(n_{0})$ into two edge orbits
$R\orbit{\alpha 0, \alpha 2}(n_{0})$ and $R\orbit{\alpha 0, \alpha 2}(n_{1})$.
Therefore the application of this rule (fig.~\ref{fig:before-after-unsew-a2})
splits the edge into two edges $H\orbit{\alpha 0, \alpha 2}(g)$ and
$H\orbit{\alpha 0, \alpha 2}(y)$.

A rule can also split an orbit implicitly. For example
(fig.~\ref{fig:rule-triangulation}) the triangulation rule splits the face orbit
$L\orbit{\alpha 0, \alpha 1}(n_{0})$ along the second implicit links. Because in
the right hand side $R\orbit{\alpha 0, \alpha 1}(n_{0})$ contains the nodes
$n_{0}$, $n_{1}$ and $n_{2}$ : the second implicit links of $n_{0}$ is deleted
and the second implicit links of $n_{1}$ and $n_{2}$ are renamed to $\alpha 2$
not in $\orbit{\alpha 0, \alpha 1}$. Therefore the application of this rule
(fig.~\ref{fig:before-after-house-triangulation}) splits
$G\orbit{\alpha 0, \alpha 1}(a)$ into four faces orbits
$H\orbit{\alpha 0, \alpha 1}(a)$, $H\orbit{\alpha 0, \alpha 1}(c)$,
$H\orbit{\alpha 0, \alpha 1}(e)$, $H\orbit{\alpha 0, \alpha 1}(g)$. Notice that
the edges of the former square face do not belong to the same face anymore.

\begin{definition}[Orbit split]
  Let $r: L \longrightarrow R$ be a rule, $m: L \to G$ a morphism,
  $t: G \longrightarrow^{r,m}H$ the transformation of G by $(r,m)$, $\orbit{o}$
  an orbit type, $n$ a preserved node and $d$ a preserved dart matched by~$n$.

  An orbit $L\orbit{o}(n)$ is \emph{splitting} by the rule $r$ either:
 \begin{itemize}
   \item explicitly if there exists two preserved nodes $n$ and $n'$ in
         $L\orbit{o}(n)$ do not belong the same orbit in $R$, i.e.
         $R\orbit{o}(n) \neq R\orbit{o}(n')$.

   \item implicitly if for all nodes within $L\orbit{o}(n)$ there exists a node
         $n'$ whose $k$-th implicit link is in $\orbit{o}$ and for all nodes
         $n'$ in $R\orbit{o}(n)$, the $k$-th implicit link is either deleted or
         renamed not in $\orbit{o}$.

   \end{itemize}
   An orbit $G\orbit{o}(d)$ is \emph{splitting} by the transformation $t$ if and
   only if there exists at least one preserved dart $d'$ in $G\orbit{o}(d)$,
   such that $H\orbit{o}(d) \neq H\orbit{o}(d')$.
\end{definition}

In order to proceed with our exemple let's start with a case of implicit split.

Given the face triangulation rule figure \ref{fig:rule-triangulation} we will
follow the implicit split event of a face of orbit type
$\orbit{\alpha 0, \alpha 1}$. The orbit $L\orbit{\alpha 0, \alpha 1}(n_{0})$
comprises node $n_{0}$ and matches a single set of darts which form a face
orbit. In this exemple nodes have two dimensions (or implicit links) in their
label. For node $n_{0}$ first implicit link is $\alpha 0$ and second implicit
link is $\alpha 1$ both in orbit type $\orbit{\alpha 0, \alpha 1}$. However the
orbit $R\orbit{\alpha 0, \alpha 1}(n_{0})$ comprises nodes $n_{0}$, $n_{1}$,
$n_{2}$ and not one of them has a second implicit link whose dimension is in
$\orbit{\alpha 0, \alpha 1}$. Under this condition we can affirm the rule figure
\ref{fig:rule-triangulation} splits a face into two or more faces. In figure
\ref{fig:house-triangulation-right-match} the orbit matches four disjointed
subsets of darts and each subset form a face orbit. The black and red dotted
cell highlights a face resulting from the split.
