\section{Conclusions}% et perspectives}
\begin{frame}{Conclusions}
    \begin{itemize}
        \item Avant toute application d'opération, méthode statique de détection des changements topologiques.
        \item Pour chaque type d'évènement, définition des conditions nécessaires et suffisantes de détection sur les règles.
        \item Cas limites clairement définis avec "faible" surcoût.
        \item Hors cas limites, aucun calcul dynamique.
        \item \color{red}{À voir les résultats de comparaisons}
    \end{itemize}
\end{frame}
