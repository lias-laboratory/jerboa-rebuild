\section{Evaluation DAG}

After the modelling of an object time and once its parametric specification has
been built, an Evaluation DAG is issued for each registered topological
parameters. %
An Evaluation DAG traces the history of a topological parameter back to the
first created orbits it originates from. %
Such a history is used in order to match the corresponding topological parameter
at re-evaluation time. %

occured in the evolution of a topological parameter. %
