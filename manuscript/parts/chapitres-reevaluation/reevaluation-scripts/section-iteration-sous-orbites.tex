\section{Structure d'itération}
\label{sec:extension-scripts-iteration}

\begin{lstlisting}[style=script-style,
language=jerboa-script,
escapechar=\%,
caption=Script de séquence d'extrusions,
label=lst:reevaluation-scripts-iteration-exemple
]
for(JerboaDart volume : <0,1,2,3>_<0,1,3>(column#0)){
    @rule<colorationFace>(face)
}
return null;
\end{lstlisting}

Les orbites étant décomposables en sous-orbites. %
La boucle \texttt{foreach} du langage de script permet donc de parcourir les
sous-orbites pour une orbite donnée. %
Prenons le script liste~\ref{lst:reevaluation-scripts-iteration-exemple}. %
Ce script contient une boucle qui pour chaque sommet (\(\orbit{1,2,3}\))
\texttt{vertex} dans une composante connexe (\(\orbit{0,1,2,3}\))
\texttt{lin\_skeleton} crée une face centrée par rapport à \texttt{vertex}. %

\subsection{Extension de la spécification paramétrique}
\label{reevaluation-scripts-iteration-extension-spec-param}

\begin{lstlisting}[style=script-style,
escapechar=\%,
caption=Spécification paramétrique avec boucle,
label=lst:reevaluation-scripts-boucle-spec-param-init]
1-creationCarre()
2-subdivisionQuad(PN%\textsubscript{0}%)
3-S_colorationColonne(PN%\textsubscript{0}%)
  For %\textlangle{}1,2,3\textrangle{}% in %\textlangle{}0,1,2,3\textrangle{}%(PN%\textsubscript{1}%)
    1-colorationFace(PN%\textsubscript{0}%)
    2-colorationFace(PN%\textsubscript{8}%)
    3-colorationFace(PN%\textsubscript{16}%)
    4-colorationFace(PN%\textsubscript{24}%)
4-triangulationFace(PN%\textsubscript{16}%)
\end{lstlisting}

Comme pour les séquences d'opérations, nous proposons d'ajouter un bloc
\texttt{For} pour lister les applications sur des sous-orbites dans un script. %
À la différence d'un bloc \texttt{Sequence}, un bloc \texttt{For} possède un
paramètre topologique pour l'orbite qui doit être itérée. %
Cela garantit que si l'orbite est supprimée, alors la boucle ne peut pas être
réévaluée. %
Sinon, si l'orbite est scindée, alors chacune des orbites issues de la scission
est réévaluée avec cette boucle. %

Détaillons l'exemple de la spécification paramétrique
liste~\ref{lst:reevaluation-scripts-boucle-spec-param-init}. %
Les deux premières applications sont les mêmes que dans la section précédente,
c'est-à-dire qu'elles produisent un chemin extrudé à partir d'un brin. %
L'application \texttt{3-\_dualToSkeleton} appelle la règle
\texttt{vertex\-Dual\-To\-Face\-Square} sur tous les sommets d'une composante
connexe, ici notre chemin extrudé qui possède quatre sommets. %
En conséquence, dans la spécification, nous voyons un bloc \texttt{For} qui
décrit quatre itérations de sommets (\(\orbit{1,2,3}\)) dans composante connexe
(\(\orbit{0,1,2,3}\)) identifiée par le PN\textsubscript{2}. %
Enfin, l'une de ces faces, celle identifiée par le PN\textsubscript{7}, est
triangulée. %

\subsection{Extension des DAGs}
\label{sec:reevaluation-scripts-iteration-extension-dags}

En plus des orbites référencées par les scripts, afin de supporter les
itérations sur les sous-orbites, il est désormais nécessaire de suivre d'autres
orbites pour conditionner la réévaluation d'un script. %
Plus précisément, il convient de tracer l'historique d'évolution de l'orbite
passée en paramètre de la boucle ainsi que les historiques des sous-orbites
itérées. %
Dans chaque DAG illustrant l'historique d'une sous-orbite itérée, il y a
à présent une branche qui trace l'historique du paramètre topologique du script,
une branche qui trace l'historique de l'orbite passée en paramètre de la
structure de contrôle et une branche qui trace l'historique pour une instance de
sous-orbite. %

\begin{lstlisting}[style=script-style,
escapechar=\%,
caption=Spécification paramétrique avec boucle,
label=lst:reevaluation-scripts-boucle-spec-param-edit]

2-subdivisionQuad(PN%\textsubscript{0}%)
3-S_colorationColonne(PN%\textsubscript{0}%)
  For %\textlangle{}1,2,3\textrangle{}% in %\textlangle{}0,1,2,3\textrangle{}%(PN%\textsubscript{1}%)
    1-colorationFace(PN%\textsubscript{0}%)
    2-colorationFace(PN%\textsubscript{8}%)
    3-colorationFace(PN%\textsubscript{16}%)
    4-colorationFace(PN%\textsubscript{24}%)
4-triangulationFace(PN%\textsubscript{16}%)
\end{lstlisting}
