
\section{Implantations}
\label{sec:reevaluation-scripts-implantation}

Ce qui est implanté~:
\begin{itemize}
  \item appariement d'orbites entre deux règles
  \item modulation de l'appariement (degré d'exactitude)
\end{itemize}

Ce qui n'est pas implanté~:
\begin{itemize}
  \item extension des specs
  \item extension des dags
\end{itemize}

Propositions~:
\begin{itemize}
  \item diagramme des specs étendues
  \item diagramme des dags étendus
\end{itemize}
