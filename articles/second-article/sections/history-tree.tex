\section{Evaluation DAG}

After an object's construction, an \emph{Evaluation DAG} (or \emph{ED}) is
issued for each PN in order to represent the construction history of all the
topological parameters designated within a Parametric Specification. %
By representing such a history, an ED makes the re-discovery of
a topological parameter possible at reevaluation time. %

An Evaluation DAG is a datastructure which lays out the events that
occured in the evolution of a topological parameter. %
