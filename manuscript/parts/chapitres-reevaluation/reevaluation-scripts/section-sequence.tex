
\section{Gestion d'une séquence}
\label{sec:extension-scripts-sequences}

  \begin{itemize}
    \item séquence \(\to\) simple succession d'opérations, pas de difficulté particulière
    \item deux options pour intégrer les scripts dans une spécification paramétrique
    \begin{itemize}
      \item boîte fermée~: on garde la spécification paramétrique simple~;
            l'utilisateur n'a aucune idée de comment le script est géré par le
            moteur de réévaluation.
      \item boîte ouverte~: la spécification paramétrique doit être restructurée
            pour afficher le structures internes des scripts~: séquences,
            itérations et alternatives.
    \end{itemize}
    \item intégration dans les DAG~: boîte qui englobe les opérations et évènements par script
  \end{itemize}
