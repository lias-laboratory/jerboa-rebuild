Aujourd'hui, les outils de modélisation géométrique sont nombreux et leurs
applications variés. %

C'est le cas, classiquement, dans le domaine de la CAO avec des outils de
modélisation paramétrique spécialisés dans la conception de pièces variées
(Solidworks), mécaniques (CATIA), de bâtiments (Revit), devant
respecter de nombreuses contraintes. %
De ce fait, ces outils permettent, entre autre, l'enregistrement des processus
de modélisation dans des spécifications paramétriques en vue de les rejouer. %

Dans le domaine de l'archéologie, chaque hypothèse avancée nécessite une
nouvelle restitution virtuelle. %
Les archéologues s'orientent donc vers des grammaires de génération procédurale
de sorte générer des variants d'un même bâtiment
\cite{Quattrini_Baleani_2015}. %
L'inconvénient de ces grammaires étant qu'elles nécessitent un riche corpus de
données, sont spécialisées et donc difficilement utilisables dans d'autres
contextes d'études. %

À ce jour, il n'existe pas, à notre connaissance, de méthode permettant le rejeu
de spécifications paramétriques à partir de règles de transformation de graphe \cite{ehrig2006fundamentals}.

En cela, l'originalité de nos travaux réside dans le fait que la méthode de
rejeu que nous concevons s'appuie sur les règles de transformations de graphe. %
Le fait que les opérations soient formalisées avec un langage de règle permet leur analyse là où analyse de code serait difficile voire impossible. %

Dans nos travaux, nous utilisons Jerboa \cite{belhaouari2014jerboa} qui est une
librairie, développée au laboratoire XLIM, permettant la construction et la
manipulation de modèles géométriques à base topologique. %
Ces modèles sont représentés avec des cartes généralisées
\cite{lienhardt1991topological,Damiand-Lienhardt14} (ou G-cartes). %

Dans cette thèse, nous poursuivons l'approche proposée par Cardot et al.
\cite{cardo2019}
