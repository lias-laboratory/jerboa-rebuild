before and after the rule's application. We can observe in figure
\ref{fig:house-triangulation-vertex-creation} a set of darts highlighted in
blue. As these darts did not exist in \ref{fig:house-triangulation-left-filter}

To statically detect the face triangulation rule creates a vertex we consider
the rule itself.
