  \begin{posterbox}[name=abstract,row=0, span=3]{Abstract}
    % \begin{minipage}[c]{0.89\linewidth}
    We extend the scope of naming problem studies to encompass rule-based graph
    transformation modeling systems. We propose a new persistent naming method
    that capitalizes on the formalized operations of generalized maps and graph
    transformation rules. It enables a unique and homogeneous characterisation
    of entities across all dimensions.

  \begin{posterbox}[name=persistent name, below=context, column=3, span=3]{Persistent naming scheme}
    Taking advantage of Jerboa graph transformation rules on generalized map's formalism:
    \begin{itemize}
      \item Using darts to represent orbits (cells)
      \item Persistent name: history of a darts' creation
    \end{itemize}
    Persistent name
    $PN_{3}=[1\fcolorbox{black}{magenta}{$n_{3}$};2\fcolorbox{black}{warmorange}{$n_{4}$};3\fcolorbox{black}{mygreen}{$n_{0}$}]$
    \begin{itemize}
      \item Rule \texttt{1-create}'s node \fcolorbox{black}{magenta}{$n_{3}$} creates dart \fcolorbox{red}{pastelpink}{3}
      \item Rule \texttt{2-extrude} matches dart \fcolorbox{red}{pastelpink}{3} and
            node \fcolorbox{black}{warmorange}{$n_{4}$} creates its copy: dart
            \fcolorbox{red}{myorange}{35}
      \item Rule \texttt{3-triangulate}'s node \fcolorbox{black}{mygreen}{$n_{0}$} matches dart \fcolorbox{red}{mygreen2}{35}
    \end{itemize}
  \end{posterbox}
