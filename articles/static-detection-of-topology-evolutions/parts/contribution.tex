\section{Event detection}

Intuitively, to detect that a rule application creates (or splits, merges, …) an
orbit, one would compare the graph before and after its application. However,
comparing the members of a rule against each other allows a static detection of
topological events. Such a method presents the advantage of being independent of
the object transformed by the rule.

A rule node matches one or more darts. So when a rule creates a node, it also
creates the corresponding darts. If the right hand side of a rule contains a
created orbit, then the application of this rule on any object creates one or
more orbits. For example, the triangulation rule
(Fig.~\ref{fig:rule-triangulation}) creates the edge orbit
$R\orbit{\alpha 0, \alpha 2}(n_{1})$. Therefore, the application of this rule on
the square face (Fig.~\ref{fig:before-after-house-triangulation}) creates four
edge orbits $H\orbit{\alpha 0, \alpha 2}(a_{1})$,
$H\orbit{\alpha 0, \alpha 2}(c_{1})$, $H\orbit{\alpha 0, \alpha 2}(e_{1})$ and
$H\orbit{\alpha 0, \alpha 2}(g_{1})$ in $H$.

\begin{definition}[Orbit creation]
  \label{def:orbit-creation}
  Let $r : L \longrightarrow R$ be a rule, $m : L \to G$ a match morphism,
  $t: G \longrightarrow^{r,m} H$ the transformation of $G$ by $(r,m)$,
  $\orbit{o}$ an orbit type, $n$ a created node of $R$ and $d$ a dart of $H$
  matched by $n$.

  An orbit $R\orbit{o}(n)$ is \emph{created} in the rule $r$ if all of its nodes
  are created in $r$.

  An orbit $H\orbit{o}(d)$ is created by the transformation $t$ if and only if
  all of its darts are created by $t$.
\end{definition}

\begin{proposition}
  Let $r : L \longrightarrow R$ be a rule, $m : L \to G$ a match morphism,
  $t: G \longrightarrow^{r,m} H$ the transformation of $G$ by $(r,m)$, $\orbit{o}$
  an orbit type, $n$ a created node of $R$ and $d$ a dart of $H$ matched by $n$.

  The orbit $H\orbit{o}(d)$ is created if and only if $R\orbit{o}(n)$ is
  created.
\end{proposition}

\begin{proofidea}
  Due to the syntactic conditions on rules, the orbit $R\orbit{o}(n)$ is
  complete. Thus, due to application of rules, $H\orbit{o}(d)$ is included in the
  image of $R\orbit{o}(n)$. Therefore, all darts of $H\orbit{o}(d)$ are created.
  Conversely, if $H\orbit{o}(d)$ is created, then it is included in the image of
  $R\orbit{o}(n)$ and then $R\orbit{o}(n)$ is created.
\end{proofidea}

Back to the example of the face triangulation
(Fig.~\ref{fig:rule-triangulation}~and~\ref{fig:before-after-house-triangulation}),
we will follow the creation of an edge. The edge orbit
$R\orbit{\alpha 0, \alpha 2}(n_{1})$ in the triangulation rule is created
because its two nodes $n_{1}$ and $n_{2}$ are created. In the transformation,
$n_{1}$ matches eight darts, including for example dart $a_{1}$, and $n_{2}$
also matches eight darts including dart $a_{2}$. The edge orbit
$H\orbit{\alpha 0, \alpha 2}(a_{1})$ is created as well as darts
$a_{2}, h_{1}, h_{2}$ (Fig.~\ref{fig:house-triangulation-right-match}).

When a rule explicitly splits an orbit, it also splits the set of matched darts.
For example, the unsewing rule (Fig.~$\ref{fig:rule-unsew-a2}$) splits the edge
orbit $L\orbit{\alpha 0, \alpha 2}(n_{0})$ into two edge orbits
$R\orbit{\alpha 0, \alpha 2}(n_{0})$ and $R\orbit{\alpha 0, \alpha 2}(n_{1})$.
Therefore the first application of this rule
(Fig.~\ref{fig:before-after-unsew-a2}) splits the edge into two edges
$H\orbit{\alpha 0, \alpha 2}(a_{0})$ and $H\orbit{\alpha 0, \alpha 2}(c_{1})$.

A rule can also split an orbit implicitly. For example the triangulation rule
(Fig.~\ref{fig:rule-triangulation}) splits the face orbit
$L\orbit{\alpha 0, \alpha 1}(n_{0})$ along the second implicit links because in
the right hand side, all the second implicit links of the nodes belonging to the
face orbit $R\orbit{\alpha 0, \alpha 1}(n_{0})$ (i.e. $n_{0}$, $n_{1}$ and
$n_{2}$) are deleted or different from $\alpha 0$ and $\alpha 1$. Therefore the
application of this rule (Fig.~\ref{fig:before-after-house-triangulation})
splits $G\orbit{\alpha 0, \alpha 1}(a_{0})$ into four face orbits
$H\orbit{\alpha 0, \alpha 1}(a_{0})$, $H\orbit{\alpha 0, \alpha 1}(c_{0})$,
$H\orbit{\alpha 0, \alpha 1}(e_{0})$, $H\orbit{\alpha 0, \alpha 1}(g_{0})$.

\begin{definition}[Orbit split]\label{def:orbit-split}
  Let $r: L \longrightarrow R$ be a rule, $m: L \to G$ a morphism,
  $t: G \longrightarrow^{r,m}H$ the transformation of G by $(r,m)$, $\orbit{o}$ an
  orbit type, $n$ a preserved node and $d$ a preserved dart matched by~$n$.

  An orbit $L\orbit{o}(n)$ is \emph{split} by the rule $r$ either:
  \begin{itemize}
	\item explicitly if there exists two preserved nodes $n$ and $n'$ in
		  $L\orbit{o}(n)$ that do not belong to the same orbit in $R$, i.e.
		  $R\orbit{o}(n) \neq R\orbit{o}(n')$.

	\item implicitly if in $L\orbit{o}(n)$ there exists a node whose $k$-th
		  implicit link is in $\orbit{o}$ and for all nodes $n'$ in
		  $R\orbit{o}(n)$, the $k$-th implicit link is either deleted or renamed
		  not in $\orbit{o}$.
  \end{itemize}

  An orbit $G\orbit{o}(d)$ is \emph{split} by the transformation $t$ if and only
  if there exists at least one preserved dart $d'$ in $G\orbit{o}(d)$, such that
  $H\orbit{o}(d) \neq H\orbit{o}(d')$.
\end{definition}

\begin{proposition}\label{prop:orbit-split}
  Let $r : L \longrightarrow R$ be a rule , $m : L \to G$ a match morphism,
  $t : G \longrightarrow^{r,m} H$ the transformation of G by $(r, m)$,
  $\orbit{o}$ an orbit type, $n$ a preserved node and $d$ a preserved dart $d$
  matched by $n$.

  The orbit $G\orbit{o}(d)$ is \emph{guaranteed} to be split if and only if
  $L\orbit{o}(n)$ is both complete on $\orbit{o}$ and split.
\end{proposition}

\begin{proofidea}\label{prfid:orbit-split}
  Since a complete orbit on $\orbit{o}$ filters all links in $\orbit{o}$, such
  an orbit $L\orbit{o}(n)$ matches all darts in $G\orbit{o}(d)$. Due to the
  application of rules, $G\orbit{o}(d)$ is included in the image of
  $L\orbit{o}(n)$ and because it is split, there exists at least one preserved
  dart $d'$ in $G\orbit{o}(d)$ such that $H\orbit{o}(d) \neq H\orbit{o}(d')$.
  Conversely, if $H\orbit{o}(d) \neq H\orbit{o}(d')$ while
  $G\orbit{o}(d) = G\orbit{o}(d')$, then $G\orbit{o}(d)$ is included in the image
  of $L\orbit{o}(n)$ and then $L\orbit{o}(n)$ is split. If the orbit to split is
  not complete then the set of matched darts can be incomplete and a split cannot
  be guaranteed.
\end{proofidea}

Back to the example of the unsewing rule (Fig.~\ref{fig:rule-unsew-a2}), we will
follow the explicit split of an edge. The edge orbit
$L\orbit{\alpha 0, \alpha 2}(n0)$ is split in the rule because its two nodes
$n_{1}$ and $n_{2}$ form two distincts edge orbits
$R\orbit{\alpha 0, \alpha 2}(n_{0})$ and $R\orbit{\alpha 0, \alpha 2}(n_{1})$.
In the transformation, $n_{0}$ (resp. $n_{1}$) matches the darts $a_{0}$ and
$b_{0}$ (resp. $c_{1}$ and $d_{1}$). The edge orbit
$G\orbit{\alpha 0, \alpha 2}(a_{0})$, comprising the darts matched by both
$n_{0}$ and $n_{1}$, is split into two orbits
$H\orbit{\alpha 0, \alpha 2}(a_{0})$ (i.e. $a_{0}$ and $b_{0}$) and
$H\orbit{\alpha 0, \alpha 2}(c_{1})$ (i.e. $c_{1}$ and $d_{1}$)
(Fig.~\ref{fig:unsew-a2-first-unsew}).

Continuing with the face triangulation rule (Fig.~\ref{fig:rule-triangulation}),
we will follow the implicit split event of a face of orbit type
$\orbit{\alpha 0, \alpha 1}$. The orbit $L\orbit{\alpha 0, \alpha 1}(n_{0})$
comprises the node $n_{0}$ and matches a single set of darts which form a face
orbit. In this exemple nodes have two dimensions (or implicit links) in their
label. For node $n_{0}$, the first implicit link is $\alpha 0$ and the second
implicit link is $\alpha 1$ both in orbit type $\orbit{\alpha 0, \alpha 1}$.
However the orbit $R\orbit{\alpha 0, \alpha 1}(n_{0})$ comprises nodes $n_{0}$,
$n_{1}$, $n_{2}$ and none of them has a second implicit link whose dimension is
in $\orbit{\alpha 0, \alpha 1}$. Under this condition we can state that the face
triangulation rule splits a face into two or more faces. Applying this
triangulation rule on the square face
(Fig.~\ref{fig:before-after-house-triangulation}),
$L\orbit{\alpha 0, \alpha 1}(n_{0})$ matches the face orbit
$G\orbit{\alpha 0, \alpha 1}(a_{0})$). Then, the square face will be split into
the four faces matched from $R\orbit{\alpha 0, \alpha 1}(n_{0})$ (i.e.
$H\orbit{\alpha 0, \alpha 1}(a_{0})$, $H\orbit{\alpha 0, \alpha 1}(c_{0})$,
$H\orbit{\alpha 0, \alpha 1}(e_{0})$ and $H\orbit{\alpha 0, \alpha 1}(g_{0})$).

Note that both the proposition~\ref{prop:orbit-split} and its proof
idea~\ref{prfid:orbit-split} mention that a split is guaranteed to happen when
the orbit is complete on $\orbit{o}$. Let us consider the unsewing rule
(Fig.~\ref{fig:rule-unsew-a2}) and its application on a surface
(Fig.~\ref{fig:before-after-unsew-a2}) once more but this time on the surface
orbit $L\orbit{\alpha 0, \alpha 1, \alpha 2}(n_{0})$. The rule nodes of this
orbit all miss an $\alpha 1$-link and, thus,
$L\orbit{\alpha 0, \alpha 1, \alpha 2}(n_{0})$ is not
$\orbit{\alpha 0, \alpha 1, \alpha 2}$-complete. Therefore, upon the first
application of the rule (Fig.~\ref{fig:unsew-a2-first-unsew}) the surface is not
split, although the conditions given in the definition~\ref{def:orbit-split} are
met. Yet, in this case, applying the unsewing rule a second time
(Fig.~\ref{fig:unsew-a2-second-unsew}) effectively splits the surface.
Conversely, when an orbit is not $\orbit{o}$-complete the matched graph is an
incomplete orbit and statically detected event cannot be guaranteed on
$\orbit{o}$. Therefore, such a statement must be confirmed dynamically.

With the same intuition as for an orbit split there are explicit and implicit
orbits merging and the logic is analogous.

For example, the sewing rule (Fig.~\ref{fig:rule-sewing-a2}) is the opposite of
the unsewing rule (Fig.~\ref{fig:rule-unsew-a2}) in regard to both the result of
its application and its construction. Let us consider the square plan
(Fig.~\ref{fig:before-after-unsew-a2}) from $H_{1}$ to $G$
(Fig.~\ref{fig:unsew-a2-first-unsew} to Fig.~\ref{fig:unsew-a2-left-match}).
There are two successives applications of the sewing rule to merge two edge
orbits $L\orbit{\alpha 0, \alpha 2}(n_{0})$ and
$L\orbit{\alpha 0, \alpha 2}(n_{1})$ into a single edge orbit
$R\orbit{\alpha 0, \alpha 2}$. Therefore, the application of the sewing rule
merges two edges of $H_{1}$ into one edge $G\orbit{\alpha 0, \alpha 2}(a_{0})$.

The same goes for the implicit merge. For example, the edge suppression rule
(Fig.~\ref{fig:rule-delete-edge}) merges the face orbits
$L\orbit{\alpha 0, \alpha 1}(n_{0})$ (i.e. the faces on both sides of the edge
orbit $L\orbit{\alpha 0, \alpha 2}(n_{0})$) along the second implicit link
because in the right hand side, the second implicit link of $n_{1}$, belonging
to the face orbit $R\orbit{\alpha 0, \alpha 1}(n_{1})$, is rewritten into
$\alpha 1$. Therefore, the application of this rule
(Fig.~\ref{fig:before-after-delete-edge}) merges
$G\orbit{\alpha 0, \alpha 1}(d_{1})$ with $G\orbit{\alpha 0, \alpha 1}(j_{1})$
into $H\orbit{\alpha 0, \alpha 1}(d_{1})$.

\begin{definition}
  Let $r : L \longrightarrow R$ be a rule, $m : L \to G$ a morphism,
  $t : G \longrightarrow^{r,m} H$ a transformation of $G$ by $(r,m)$,
  $\orbit{o}$ an orbit type, $n$ a preserved node and $d$ a preserved dart
  matched by $n$.

  An orbit $L\orbit{o}(n)$ is merged by the rule $r$ either:
  \begin{itemize}
	\item explicitly if there exists a preserved node $n'$, with $n \neq n'$
		  and $L\orbit{o}(n) \neq L\orbit{o}(n')$, such that $n$ and $n'$ belong
		  to the same orbit in $R$, i.e. $R\orbit{o}(n) = R\orbit{o}(n')$.

	\item implicitly if in $R\orbit{o}(n)$, there exists a node whose $k$-th
		  implicit link is in $\orbit{o}$ and for all nodes $n'$ in
		  $L\orbit{o}(n)$, the $k$-th implicit link is not in $\orbit{o}$.
  \end{itemize}

  Two orbits $G\orbit{o}(d)$ and $G\orbit{o}(d')$, with $d'$ a preserved dart,
  are merged by the transformation $t$ if and only if
  $H\orbit{o}(d) = H\orbit{o}(d')$.
\end{definition}
