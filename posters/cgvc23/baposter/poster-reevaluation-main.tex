  \begin{posterbox}[name=abstract,row=0,span=2]{Abstract}
    \begin{minipage}[c]{0.89\linewidth}
    We extend the scope of naming problem studies to encompass
    rule-based graph transformation modeling systems. We propose a novel
    persistent naming method that capitalizes on the formalized operations of
    generalized maps and graph transformation rules. It enables a unique and
    homogeneous characterisation of entities across all dimensions
    \end{minipage}

  \begin{posterbox}[name=persistent name, below=context, column=1]{Persistent naming scheme}
    Taking advantage of Jerboa graph transformation rules on generalized map's formalism:
    \begin{itemize}
      \item Using darts to represent orbits (cells)
      \item Persistent name: history of a darts' creation
    \end{itemize}
    Persistent name $PN_{3}=[1{\color{green}n_{3}};2{\color{green}n_{4}};3{\color{green}n_{0}}]$
    \begin{itemize}
      \item Rule \texttt{1-create}'s node {\color{green}$n_{0}$} creates dart {\color{red}3}
      \item Rule \texttt{2-extrude}'s node {\color{green}$n_{1}$} matches dart
            {\color{red}3}, node {\color{green}$n_{4}$} creates dart {\color{red}35}
      \item Rule \texttt{3-triangulate}'s node {\color{green}$n_{0}$} matches dart {\color{red}35}
    \end{itemize}
  \end{posterbox}
